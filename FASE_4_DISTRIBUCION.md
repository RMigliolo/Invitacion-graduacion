# FASE 4: Distribución de la invitación

## Enlace oficial de la invitación

https://rmigliolo.github.io/Invitacion-graduacion/

## Recomendación sobre enlace corto

Para evitar costos y mantener estabilidad, la opción principal recomendada es usar el enlace directo de GitHub Pages. Ya es suficientemente claro, gratuito y no depende de servicios externos.

Si deseas un enlace todavía más corto, puedes crear uno gratis en TinyURL o Bitly. Se recomienda usarlo solo como complemento para mensajes, no como enlace principal del QR, porque el QR debe depender del enlace más estable posible.

## Cambios técnicos agregados al HTML

Se añadieron metadatos Open Graph y Twitter Card para que al compartir el enlace se muestre una vista previa más profesional en redes sociales y aplicaciones de mensajería.

## Uso recomendado

1. Para WhatsApp: enviar el texto preparado en `TEXTOS_DISTRIBUCION.md` junto con el enlace directo.
2. Para estados/historias: usar `historia-whatsapp-instagram.png`.
3. Para Facebook o Instagram: usar `post-instagram-facebook.png`.
4. Para impresión: usar `qr-invitacion-elegante.pdf` o `qr-invitacion-elegante.png`.
